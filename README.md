# Masterstroke Landing Page

Premium, responsive marketing website for **Masterstroke**, featuring three products:

- TiltFlow Roller
- Pressurised Paint Roller with Standardised Lid
- 2-in-1 Sander + Smoothener

Built with plain HTML/CSS/JS (no build tooling required).

## Run locally

From this folder:

```bash
python3 -m http.server 4173
```

Open in your browser:

- http://localhost:4173
- http://127.0.0.1:4173

## Files

- `index.html` – page content and product sections
- `styles.css` – colourful premium design, responsive layout, animations
- `script.js` – menu interactions, reveal animations, footer year, CTA feedback

## Notes

- No dependency installation required.
- Works as a static site on any basic web server.
